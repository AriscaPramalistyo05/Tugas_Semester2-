Spread Operator

Spread ini digunakan untuk menyebarkan atau menggabungkan nilai aray atau lebih. Spread operator dituliskan dengan tiga titik (...).

  • Data array nama : ["Listyo", " Erwin ", "Azka "]
  • Data array prodi : ["RPL"," ABI "," AKP"]
  • Memanggil menggunakan console.log(...mahasiswa)
