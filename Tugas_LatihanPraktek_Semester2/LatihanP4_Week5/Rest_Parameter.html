Rest Parameter


    

    

      Rest Parameter digunakan untuk mengakomodasi jumlah argumen yang tidak
      terbatas dalam sebuah fungsi.
    

    

      Contoh ini menjumlahkan sejumlah angka yang diberikan sebagai argumen saat
      memanggil fungsi "ram". Output akan ditampilkan pada halaman web setelah
      tombol "Click On" ditekan.
