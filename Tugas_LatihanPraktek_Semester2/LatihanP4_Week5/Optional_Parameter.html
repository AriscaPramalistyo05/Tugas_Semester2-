Optional Parameter

Konversi Suhu

Optional parameter dalam JavaScript memungkinkan pengguna untuk mengabaikan atau tidak mengisi nilai saat memanggil fungsi, dengan memberikan nilai default kepada parameter tersebut untuk digunakan jika tidak diisi. Hal ini memberikan fleksibilitas dalam penggunaan fungsi, terutama ketika terdapat parameter yang bersifat opsional atau tidak selalu diperlukan dalam setiap situasi.
